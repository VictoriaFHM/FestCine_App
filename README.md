# FestCine App

Sistema de gestión para un Festival de Cine Independiente.

## Tecnologías usadas

- Node.js
- Express
- SQL Server
- HTML
- CSS
- JavaScript

## Ejecución del proyecto

1. Ejecutar los scripts SQL de la carpeta SQL en SQL Server.
2. Instalar dependencias:

npm install

3. Ejecutar el servidor:

node server.js

4. Abrir en el navegador:

http://localhost:3000

## Importante

Si el servidor SQL no es (localdb)\MSSQLLocalDB, modificar la conexión en db.js.
